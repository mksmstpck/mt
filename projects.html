Проекти

PogodkaPyBot

Один із моїх проєктів — це телеграм-бот, який повідомляє користувачеві актуальну погоду у зручному форматі. Я розробив його як приклад практичного застосування програмування, де поєднується робота з API, обробка повідомлень і форматування відповідей. Бот вміє визначати погоду за вказаним містом, показує температуру, вологість, вітер та короткий опис стану погоди. Його можна використовувати як частину навчального прикладу, щоб показати учням, як програмування може вирішувати реальні задачі. У процесі створення я використав мову програмування Python та бібліотеку python-telegram-bot для взаємодії з Telegram API, а також OpenWeatherMap API для отримання метеоданих. Особливу увагу приділив зручності користування: бот не вимагає складних команд, працює швидко і реагує на прості запити. Цей проєкт допоміг мені не лише прокачати навички роботи з API та мережевими запитами, а й зрозуміти, як створювати інтерфейс, зручний для звичайного користувача. У майбутньому планую розширити його можливості — додати прогноз на кілька днів, геолокацію та підтримку кількох мов.

MovieGarden

Один із моїх проєктів — це сайт для перегляду фільмів, де я реалізував повноцінну backend-частину за допомогою мови Go (Golang). Метою було створити простий, швидкий і зручний сервіс, який дозволяє користувачам переглядати фільми онлайн, зберігати обране та залишати відгуки. Сайт має авторизацію, систему користувачів, базу даних для зберігання інформації про фільми та інтеграцію з API для завантаження описів, постерів і рейтингів. Backend написано з використанням Gorilla Mux для маршрутизації, GORM для роботи з базою даних та JWT для аутентифікації. Я зробив акцент на швидкості відповіді сервера, правильній структурі проєкту та зручному REST API, який у майбутньому легко масштабувати. Цей проєкт дав мені глибше розуміння побудови веб-сервісів на Go та особливостей роботи з потоками, кешуванням і безпекою. Працюю над тим, щоб додати адаптивну фронтенд-частину та фільтрацію фільмів за жанрами, рейтингом і новизною.
